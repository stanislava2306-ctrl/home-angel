Политика конфиденциальности

Главное — коротко

  • Ответы на вопросы теста, результаты, история прохождений, план и дневник хранятся только в вашем браузере (localStorage) и не отправляются на сервер и третьим лицам.
  • Единственное исключение — платный ИИ-разбор: при его заказе ответы передаются по защищённому соединению на сервер функции сайта и в Anthropic API исключительно для формирования текста разбора. Ни сайт, ни его серверные функции не сохраняют эти данные.
  • Оплата обрабатывается сервисом ЮKassa; сайт не получает и не хранит данные вашей карты.
  • Сайт не использует рекламные трекеры и не собирает персональные данные для аналитики.

Удаление данных

Все данные теста можно удалить, очистив данные сайта в настройках браузера (localStorage). После этого восстановить их невозможно.
