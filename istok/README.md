# Исток — тест на детские паттерны

Один самодостаточный файл `index.html`: тест на 50 вопросов, скоринговый движок,
страница результатов, ежедневная проработка и инфоблок со статьями. Звуковые
иллюзии синтезируются в браузере, зрительные — рисуются кодом, поэтому сайт
работает офлайн и без внешних зависимостей.

## Как открыть

- Локально: просто открыть `index.html` в браузере.
- GitHub Pages: после мержа в `main` — `https://<логин>.github.io/home-angel/istok/`.
- Netlify: перетащить папку `istok` в Netlify Drop, или подключить репозиторий
  и указать **Base directory: `istok`**.

## ИИ-разбор (кнопка «Получить ИИ-разбор»)

Работает только на Netlify: `netlify/functions/analyze.mjs` — серверная функция,
которая обращается к Claude API. Чтобы включить:

1. Задеплоить сайт на Netlify (base directory `istok`).
2. В настройках сайта → Environment variables добавить
   `ANTHROPIC_API_KEY` = ключ с console.anthropic.com.
3. (Необязательно) `ISTOK_MODEL` — id модели, по умолчанию `claude-opus-4-8`.

Ключ никогда не попадает в браузер. На GitHub Pages и локально кнопка
показывает понятное сообщение и не ломает остальную страницу.

## Свои ассеты (необязательно)

См. `assets/README.md`: `assets/music.mp3` заменит генеративную музыку,
`assets/cards/*.jpg` заменят рисованные карточки блока 5.
