# Ассеты Истока (необязательные)

Тест полностью работает без этой папки: звуки синтезируются в браузере,
иллюстрации рисуются кодом. Но сюда можно положить свои файлы — код
подхватит их автоматически.

## Фоновая музыка

Положи файл **`music.mp3`** прямо в эту папку (`istok/assets/music.mp3`) —
и он заменит генеративную мелодию. Подойдёт любой спокойный эмбиент-трек
с бесплатного стока (Pixabay Music, Free Music Archive, Chosic — фильтр
«ambient / meditation», лицензия CC0 или Royalty Free). Файл зациклится сам.

## Фото для карточек блока 5 («Кем ты себя видишь?»)

Положи фотографии в `istok/assets/cards/` с такими именами:

| Файл | Карточка |
|---|---|
| `cards/bird.jpg` | Птица в клетке |
| `cards/tree.jpg` | Дерево на ветру |
| `cards/stone.jpg` | Камень в воде |
| `cards/candle.jpg` | Свеча |

Рекомендуемый размер ~800×500 (горизонтальные). Пока файла нет,
показывается нарисованная сцена; как только файл появится — покажется фото.
